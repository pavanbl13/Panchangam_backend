  git push -u origin main

METHOD 2: Using HTTPS with Personal Access Token (PAT)
======================================================
  1. Create Personal Access Token on GitHub:
     - Go to Settings → Developer settings → Personal access tokens
     - Click "Generate new token (classic)"
     - Check: repo, read:user, write:repo_hook
     - Copy the token

  2. Push with token:
     git push -u origin main
     (When prompted for password, use the PAT as password)

METHOD 3: Using SSH (Requires SSH Key Setup)
=============================================
  1. Set up SSH key on GitHub first
  2. Update remote URL:
     git remote set-url origin git@github.com:pavanbl13/Panchangam_backend.git
  3. Push:
     git push -u origin main
